# sports-scores

MLB scoreboard using Angular

### Data Source
The data source is a deprecated MLB Score API found on the MLB Hackathon site: http://gd.mlb.com/components/game/mlb

 ### Tools
* Angular

 ### Running
* Navigate into the client folder
```
$ cd client
```
* Install Dependencies:
```
$ npm install
```
* Run App:
```
$ ng serve
```
